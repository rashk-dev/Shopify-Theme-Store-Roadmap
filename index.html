00 Setup Commands
Shopify CLI Setup Setup
shopify theme commands

Initialize your theme project using Shopify CLI. This creates the foundation for your development workflow and connects to your store.

  • Install Shopify CLI globally on your machine
  • Navigate to your project folder
  • Run: shopify theme init
  • cd storename
  • code .
  • shopify theme dev
shopify theme init shopify theme dev
Git Initialize & Push Setup
git init → git push origin main

Initialize Git in your project folder and set up version control. Follow these commands in order to prepare your theme for GitHub.

  • Make the Repo, keep it private and no Readme file
  • Initialize Git: git init
  • Create .gitignore file in root folder (echo ".DS_Store\nnode_modules/\n*.log\nconfig.yml" > .gitignore)
  • Add node_modules/, .env, and other sensitive files to .gitignore
  • Stage all files: git add .
  • Make first commit: git commit -m "Initial theme setup"
  • Set main branch: git branch -M main
  • Connect to GitHub: git remote add origin [your-repo-url]
  • Push to GitHub: git push -u origin main
  • git add . || git commit -m "describe your changes" || git push
git init git add . git commit -m git push origin main
01 Foundation
Locales Foundation
locales/en.default.json

Translation keys for all customizer labels and UI text. Every t: key in schema must have a matching entry here.

  • Add general, labels, options translation keys
  • Add cart, search, password, 404 text strings
  • Ensure all t: keys match settings_schema.json
locales/en.default.json
Settings Schema Foundation
config/settings_schema.json

Defines every setting in Theme Settings panel. Colors, fonts, spacing, buttons, nav, footer. Each id becomes a settings.id Liquid variable.

  • Typography — primary + secondary font pickers
  • Layout — page width + margin settings
  • Colors — background, foreground, accent, secondary
  • Buttons — color, text color, border radius
  • Navigation — header bg + text color
  • Footer — footer bg + text color
config/settings_schema.json
CSS Variables Foundation
snippets/css-variables.liquid

Bridges customizer → CSS. Outputs all settings as :root CSS variables. Also defines spacing scale and typography scale for consistent design.

  • Map all color settings to --color- variables
  • Map font settings to --font- variables
  • Add spacing scale: --space-xs to --space-2xl
  • Add typography scale: --text-sm to --text-5xl
  • Add button and input radius variables
snippets/css-variables.liquid
Critical CSS + theme.liquid Foundation
assets/critical.css + layout/theme.liquid

theme.liquid is the master HTML shell that wraps every page. critical.css loads on every page — contains resets, body styles and the section grid layout system.

  • CSS reset — box-sizing, margins, media elements
  • Body font, background, color from CSS variables
  • Section grid system — .shopify-section layout
  • .full-width utility class for edge to edge sections
  • Verify content_for_header and content_for_layout in theme.liquid
assets/critical.css layout/theme.liquid

06 Launch
Push to GitHub Deploy
git init → git push origin main

Push theme to GitHub for version control. Connect to Shopify for auto-deploy on every push — professional workflow.

  • git init (if not already done)
  • git add . && git commit -m "Initial theme"
  • Create repo on GitHub and push
  • Admin → Themes → Add theme → GitHub

Publish 🚀 Deploy
shopify theme push → Admin → Publish

The final step. Push the theme and publish it live. The store goes live with your custom theme!

  • shopify theme push --store your-store.myshopify.com
  • Admin → Online Store → Themes
  • Find your theme → click Publish
  • Celebrate! 🎉
